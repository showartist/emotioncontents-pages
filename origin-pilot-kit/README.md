# ORIGIN PILOT KIT · 검증엔진

GitHub Pages에 바로 올릴 수 있는 정적 웹앱입니다.

## 목적
이 앱은 원천 IP를 예쁘게 정리하는 데서 끝나지 않고,

1. **핵심 검증 질문**을 만들고
2. **가설별 성공 기준**을 설정하고
3. **실험 프로토콜**을 제안하고
4. **실제 사용자 응답**을 기록하고
5. **GO / ITERATE / PIVOT / STOP** 판단을 업데이트합니다.

즉, 생성기보다 **검증엔진**에 가깝게 설계했습니다.

## 포함 기능
- 사주보이즈 / 공연형 샘플 자동 로드
- IP 브리프 입력
- 검증 가설 자동 생성
- 실험 설계 자동 생성
- 사용자 응답 로그 입력
- 지표 집계
  - 첫 인식 적중률
  - 재방문 YES 비율
  - 차별감 / 명확성 평균
  - 행동 전환 신호
  - 혼란 응답 비율
- 현재 판단 자동 계산
- Markdown 리포트 저장
- JSON 상태 내보내기 / 불러오기
- localStorage 자동 저장

## 배포 위치
이 폴더를 그대로 GitHub Pages 저장소에 올리면 아래 경로로 접속 가능합니다.

```text
https://showartist.github.io/emotioncontents-pages/origin-pilot-kit/
```

## 기술 제약
- 백엔드 없음
- 외부 API 없음
- GitHub Pages 호환 정적 구조

## 왜 이 구조인가
해커톤/캡스톤 초기에는 “AI가 IP를 성공 판정한다”는 주장보다,
**AI가 검증 가능한 실험으로 바꿔준다**는 구조가 더 현실적이고 설득력 있습니다.
